🚀 Quick Start

  1. Run supabase_schema.sql in Supabase SQL Editor
  2. Settings → fill LiveKit, Gemini, Vobiz → Save
  3. Click ⚡ Create SIP Trunk in Settings → Vobiz
  4. AI Prompt → customise for your business → Save
  5. Agents → create at least one profile
  6. Single Call → test with your own number
